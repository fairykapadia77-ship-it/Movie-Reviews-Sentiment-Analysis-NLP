# IMDb Movie Reviews Sentiment Analysis (NLP)

## 📌 Project Overview
This project performs text analytics and Natural Language Processing (NLP) on a large dataset of movie reviews. By leveraging lexical analysis, the project automates the classification of user reviews into positive, negative, or neutral categories, providing structured insights from unstructured text data.

## 🛠️ Tech Stack & Tools
- **Language:** Python
- **NLP Library:** TextBlob
- **Data & Visualization:** Pandas, Seaborn, Matplotlib

## 🔍 Key Analysis Performed
1. **Lexical Sentiment Scoring:** Used `TextBlob` to calculate the polarity score of text reviews, scaling from -1 (extremely negative) to +1 (extremely positive).
2. **Behavioral Classification:** Categorized reviews based on numeric distribution thresholds to evaluate overall user reception.
3. **Distribution Analytics:** Visualized the final sentiment metrics using count plots to observe the density of positive vs. negative reviews.

## 📈 Business Insights & Actions
- **Automated Market Research:** Demonstrates how businesses can automatically monitor customer feedback, product reviews, or social media mentions at scale without manual reading.
- **Unstructured Data Mastery:** Proves the ability to parse, clean, and extract quantitative value from messy text blocks, a highly valued skill in modern data organizations.
